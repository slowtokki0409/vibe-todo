## 1️⃣ CRUD 기본 동작 검증

### 📝 Create (생성)

**테스트 시나리오**:
1. "프로젝트 제안서 작성" 입력 후 Enter → 기본 우선순위(중간), 카테고리(개인)로 생성
2. "건강검진 예약하기" 입력 → 고급 옵션으로 우선순위(높음), 카테고리(업무) 설정 후 생성
3. "디자인 시스템 업데이트" 입력 후 Enter → 기본값으로 생성
4. "건강검진 예약하기" 입력 → 우선순위(낮음), 카테고리(학습) 설정 후 생성

**결과**:
- ✅ 입력 즉시 리스트에 추가됨
- ✅ Framer Motion stagger 애니메이션 정상 작동
- ✅ 고급 옵션 패널 토글 정상
- ✅ 우선순위/카테고리 배지 정확히 표시
- ✅ 통계 카드 실시간 업데이트 (0/0 → 0/4개)

**점수**: 25/25

---

### 📖 Read (조회)

**테스트 시나리오**:
- 4개 항목이 우선순위순으로 정렬되어 표시되는지 확인
- 통계 카드가 정확한 수치를 보여주는지 확인

**결과**:
- ✅ 높음(2개) → 중간(1개) → 낮음(1개) 순으로 정렬
- ✅ 통계 카드 정확:
  - 전체 진행률: 0% (완료 0개 / 전체 4개)
  - 완료된 작업: 0개 (4개 중 0개)
  - 남은 작업: 4개
  - 기한 초과: 0개
  - 높은 우선순위: 2개
- ✅ 각 항목의 배지(우선순위, 카테고리) 정확히 표시

**점수**: 부여 없음 (Create에 포함)

---

### ✏️ Update (수정)

**테스트 시나리오**:
- "디자인 시스템 업데이트" 항목의 체크박스 클릭 → 완료 처리

**결과**:
- ✅ CheckCircle2 아이콘으로 변경
- ✅ 텍스트 line-through + 회색 처리
- ✅ 애니메이션 전환 부드러움 (0.3s duration)
- ✅ 통계 업데이트:
  - 전체 진행률: 0% → 25%
  - 완료된 작업: 0개 → 1개
  - 남은 작업: 4개 → 3개

**점수**: 부여 없음 (Create에 포함)

---

### 🗑️ Delete (삭제)

**테스트 시나리오**:
1. 완료된 항목("디자인 시스템 업데이트")에 hover
2. 삭제 버튼(쓰레기통 아이콘) 나타남 확인
3. 클릭하여 삭제

**결과**:
- ✅ Hover 시 opacity 0 → 1 전환 (group-hover)
- ✅ 클릭 시 exit 애니메이션 (y: -20, scale: 0.9)
- ✅ 리스트에서 즉시 제거
- ✅ 통계 업데이트:
  - 전체 진행률: 25% → 0% (완료 항목 삭제됨)
  - 완료된 작업: 1개 → 0개
  - 전체 개수: 4개 → 3개

**점수**: 10/10 (별도 항목)
